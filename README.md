# FreSpeD
Frequency specific change point detection (<tt>FreSpeD</tt>) 

## Installation

You can install the development version of <tt>FreSpeD</tt> from
[GitHub](https://github.com) with:

``` r
devtools::install_github("HoussamBoukhe/FreSpeD")
```

## Dependencies
R version: 4.4.1

doParallel 1.0.17
foreach 1.5.2
reshape 1.4.4
ggplot2 3.5.1
